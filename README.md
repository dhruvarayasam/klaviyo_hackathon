Klaviyo Hackathon Entry

Self Healing Campaigns
Adjusts campaigns on the fly by using webhooks to monitor performance and generates possible segmentations and campaign refactors for increased engagement.

- Uses Klaviyo MCP server to interface with metrics data, as well as interact with campaign and flows information. 
- Creates new campaign suggestions and can push directly to Klaviyo through API integration based on metric performance and customer segmentation.
